# tornado-basic

This project has been inspired by https://www.youtube.com/watch?v=4xGQS8Pv4io to experiment with nginx as a load balancer.

Setup virtual environment

```
python3 -m venv .venv
pip install -r requirements.txt
```

Spin up the server

```
. .venv/bin/activate
python index.py 1111
```
